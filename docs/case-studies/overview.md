# Real-World Case Studies

## Purpose of This Section

These case studies present realistic (composite/illustrative, not real customer-identifying) scenarios that synthesize multiple AML/KYC concepts to test applied understanding. Each case integrates red flag identification, investigative methodology, and decision-making.

## Available Case Studies

- [Merchant Fraud Investigation](/docs/case-studies/merchant-fraud)
- [Trade-Based Money Laundering](/docs/case-studies/trade-based-ml)
- [Cryptocurrency Laundering](/docs/case-studies/crypto-laundering)
- [PEP Investigation](/docs/case-studies/pep-investigation)

## How to Use These Case Studies

1. Read the scenario and try to identify red flags independently before reading the analysis
2. Consider what investigative steps you would take
3. Compare your approach to the model analysis provided
4. Consider how the case would differ under different jurisdictional frameworks
