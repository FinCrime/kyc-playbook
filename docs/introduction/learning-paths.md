## Quick Reference: Which Path Is Right for Me?

| I am... | Recommended Path |
|---------|-----------------|
| A student with no compliance experience | Path 1: Foundation |
| A fresh graduate joining an AML team | Path 1 → Path 2 |
| A KYC analyst with 1–2 years experience | Path 2: Investigation Analyst |
| A senior analyst targeting a QA role | Path 3: Senior & QA |
| Preparing for job interviews | Path 2 + [Interview Prep](/docs/career/interview/aml-analyst) |
| Studying for CAMS exam | Path 4: CAMS |
| Moving into merchant risk/KYB | Path 3, Module 2 |
