# KYC Checklist — Quick Reference

## Individual Customer Onboarding Checklist

### Identity (CIP)
- [ ] Government-issued photo ID collected (passport / national ID / driver's licence)
- [ ] ID verified: not expired, photo matches, security features intact
- [ ] Biometric/liveness check completed (where applicable)
- [ ] ID details match across all submitted documents

### Address Verification
- [ ] Proof of address collected (utility bill / bank statement / government letter)
- [ ] Document dated within 3 months
- [ ] Address consistent with other application information

### Screening
- [ ] Sanctions screening completed (OFAC, UN, EU, UK, domestic lists)
- [ ] PEP screening completed
- [ ] Adverse media search conducted and findings documented

### Risk Assessment
- [ ] Customer risk factors assessed (geography, occupation, source of wealth)
- [ ] Risk rating assigned (Low / Medium / High) with documented rationale
- [ ] Appropriate due diligence level determined (SDD / CDD / EDD)

### CDD (Standard)
- [ ] Business purpose of relationship documented
- [ ] Expected transaction profile documented
- [ ] Source of funds documented (proportionate to risk)

### EDD (If High Risk)
- [ ] EDD trigger documented
- [ ] Enhanced identity verification completed
- [ ] Source of wealth documented with independent evidence
- [ ] Adverse media deep dive completed
- [ ] Senior management approval obtained
- [ ] Enhanced monitoring plan established

## Entity (Corporate) Onboarding Checklist

### Entity Verification
- [ ] Certificate of incorporation verified (registry check)
- [ ] Entity status: active / good standing confirmed
- [ ] Business activity consistent with stated purpose
- [ ] Registered address verified (not mass-incorporation agent)
- [ ] Required licenses verified (if regulated activity)

### Ownership & Control
- [ ] Register of directors collected
- [ ] Register of shareholders collected
- [ ] UBO identified (≥25% ownership threshold, or control-based)
- [ ] Ownership calculation documented if indirect ownership
- [ ] UBO declaration signed by customer

### Documentation
- [ ] Memorandum & Articles of Association collected
- [ ] UBO identity documents collected for all UBOs
- [ ] UBOs screened: sanctions, PEP, adverse media

### Risk Assessment
- [ ] Entity risk rated
- [ ] Industry / geographic risk factored in
- [ ] If high risk: EDD triggered (see EDD checklist)

---

## Periodic Review Checklist

- [ ] Existing documentation reviewed for currency (IDs, address proofs)
- [ ] Out-of-date documents identified and new ones requested
- [ ] Re-screening performed (sanctions / PEP / adverse media) with current data
- [ ] Transaction history since last review assessed
- [ ] Material changes in profile identified and acted upon
- [ ] Risk rating confirmed or revised (with rationale)
- [ ] Review date recorded; next review date set

---

## Quick Reference: Document Validity Periods

| Document | Typical Maximum Age |
|---|---|
| Proof of address (utility bill, bank statement) | 3 months |
| Government-issued photo ID | Must be valid (not expired) |
| Company certificate of good standing | Often 3–6 months (varies by institution) |
| Financial statements | Most recent annual statements |
