## What Is KYC Playbook?

KYC Playbook is a free, open-source educational platform for compliance professionals, students, and career-switchers learning AML, KYC, EDD, sanctions screening, and financial crime investigation.

Built by practitioners. For practitioners.

## What's Covered

- **AML Fundamentals** — Money laundering stages, typologies, terrorist financing
- **KYC / CDD / EDD** — Full due diligence framework from CIP to Enhanced DD
- **KYB & Merchant Risk** — Business verification, UBO identification, payment gateway MDD
- **Sanctions Screening** — OFAC, UN, EU, UK sanctions; 50% Rule; hit resolution
- **PEP & Adverse Media** — Investigation methodology, RCA identification
- **OSINT Techniques** — Google dorks, business registries, WHOIS, court records
- **Transaction Monitoring** — Alert management, investigation, red flags
- **SAR/STR Filing** — Multi-jurisdiction requirements, narrative writing
- **Practice Labs** — Hands-on investigation exercises with model solutions
- **Career & Interview Prep** — 500+ curated Q&As for AML/KYC roles
- **CAMS Preparation** — Structured study guide with practice questions
- **Glossary, Cheat Sheets & Templates** — Daily reference materials

## Quick Start

```bash
# Clone the repository
git clone https://github.com/FinCrime/kyc-playbook.git
cd kyc-playbook

# Install dependencies
npm install

# Start local development server
npm run start

# Build for production
npm run build
```
